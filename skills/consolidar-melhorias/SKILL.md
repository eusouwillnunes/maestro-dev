Consolide as melhorias pendentes do Sistema Maestro.

Use a skill `revisao-sistema` para executar a consolidação.

Passos:
1. Busque arquivos `melhorias-sistema-maestro.md` nos projetos recentes
2. Agrupe todas as observações por agente
3. Ordene por prioridade (Alta primeiro)
4. Apresente o resumo: quantas observações por agente
5. Pergunte por qual agente o usuário quer começar
6. Quando o usuário escolher, proponha as alterações no SKILL.md do agente
7. Após aprovação, gere o arquivo atualizado para download
